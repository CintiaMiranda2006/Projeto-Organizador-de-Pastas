# PRD — Painel de Administração

## Visão geral

Especificação do painel de controle para administradores do sistema.

## Funcionalidades

- Listagem de todos os usuários cadastrados.
- Busca por nome ou email.
- Ativação e desativação de contas.
- Exportação de relatório de usuários em CSV.

## Regras de negócio

- Apenas usuários com perfil `admin` têm acesso ao painel.
- Ações de ativação/desativação devem gerar log com data e responsável.
- A exportação deve incluir apenas dados não sensíveis.

## Critérios de aceite

- Admin visualiza lista de usuários com paginação.
- Busca retorna resultados em tempo real.
- CSV exportado é aberto corretamente no Excel.
